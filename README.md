## About

This repository is a private learning archive and educational documentation space. It is used to organize completed courses, certificates of completion, platform badges, learning notes, lab work, roadmaps, and study resources in a way that can scale as more material is added.

The repository is not meant to be only a badge gallery. Badges remain preserved as visual proof of learning progress, while the folder structure gives each topic a dedicated place for certificates, notes, labs, roadmaps, and useful references.

---

## Archive Structure

```text
.
|-- Learning-Archive/
|   |-- AI/
|   |-- AWS/
|   |-- Backend/
|   |-- Certificates/
|   |-- Cybersecurity/
|   |-- Databases/
|   |-- Labs/
|   |-- Learning-Notes/
|   |-- Microsoft/
|   |-- Misc/
|   |-- NetAcad/
|   |-- Resources/
|   |-- Roadmaps/
|   `-- Soft-Skills/
`-- assets/
    |-- badges/
    |-- generated/
    |-- readme/
    `-- static/
```

Each folder inside `Learning-Archive` can hold its own `README.md`, certificate references, notes, labs, and resource links. Shared badge assets remain in `assets/badges` so existing visual assets stay stable and reusable.
