# NetAcad

Cisco Networking Academy learning archive for programming essentials, networking, cybersecurity, Packet Tracer, and data science foundations.

## Focus

- C++ essentials and advanced C++
- Python essentials
- HTML, CSS, and JavaScript essentials
- Networking basics
- Introduction to cybersecurity
- Packet Tracer practice
